## Shared Dataset

The OWID COVID-19 dataset is not duplicated inside every assignment folder. Download it once from the `CSE468-information-visualization/` directory when needed:

```bash
curl -L --create-dirs -o data/owid-covid-data.csv https://covid.ourworldindata.org/data/owid-covid-data.csv
```

Visualizations that use COVID-19 data read from `data/owid-covid-data.csv`.

# CSE271 Principles of Programming Languages

## Setting Up

1. Install [Racket](https://racket-lang.org/download/).
2. [Optional] Install [Visual Studio Code](https://code.visualstudio.com/download). You can also use any other text editor of your choice.
3. [Optional] Install a Racket extension for Visual Studio Code. This can be handy for running Racket code from Visual Studio Code.
